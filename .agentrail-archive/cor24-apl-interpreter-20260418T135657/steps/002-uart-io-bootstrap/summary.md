Implemented UART I/O bootstrap: io_getline() with backspace/DEL support (BS-SPACE-BS visual erase), io_print() for prompt output. Main.c wired as echo REPL with COR24 APL v0.1 banner, 6-space APL-style prompt, and line echo. Tested banner output, character echo, backspace editing, and empty line handling via cor24-run --terminal. tc24r stdio.h already provides putchar/getchar/puts/printf, so io.h adds only the line-editing layer on top.
