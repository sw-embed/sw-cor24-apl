Added parse_int() and print_int() in src/num.h with APL underscore-negative convention. parse_int reads decimal integers from string with underscore prefix for negatives. print_int outputs with same convention. REPL updated to parse numeric input and round-trip it (normalizes leading zeros, _0→0). Non-numeric input falls through to text echo. Verified: 42, _7, 0, _123, hello, 100, 007→7, _0→0.
