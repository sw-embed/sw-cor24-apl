Added vector literal support via number stranding. Adjacent NUM tokens in parser are collected into heap-allocated vectors (NODE_VEC). Evaluator transitioned from raw scalar returns to heap array indices -- all values are now arrays. Variables store heap indices enabling vector assignment/recall. REPL uses heap mark/restore for per-iteration temporary reclamation. Verified: 1 2 3 -> vector, _1 _2 _3 -> negative vector, V <- 1 2 3 then V -> recall, scalar arithmetic still works, assignments silent.
