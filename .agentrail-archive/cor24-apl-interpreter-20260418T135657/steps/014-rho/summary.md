Implemented monadic rho (shape-of) returning empty vector for scalars, scalar length for vectors, 2-element shape for matrices. Implemented dyadic rho (reshape) with cyclic fill supporting vector and matrix results. Added NODE_DYAD AST type and parser support for dyadic reserved words after left arguments. Tested: rho 5 -> empty, rho 1 2 3 -> 3, 3 rho 7 -> 7 7 7, 5 rho 1 2 3 -> 1 2 3 1 2, rho 2 3 rho iota 6 -> 2 3.
